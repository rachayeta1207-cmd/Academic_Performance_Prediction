Project Title
Academic Performance Prediction using Machine Learning

Description
This project predicts student academic performance using a Machine Learning classification model.
A Flask web application allows users to enter study hours, attendance, and previous scores to get a prediction.

Technologies Used
Python
Scikit-learn
Flask
Pandas
NumPy
HTML/CSS

How to Run the Project
1. Install required libraries
pip install -r requirements.txt

2. Run the application
python app.py

3. Open browser
http://127.0.0.1:5000
